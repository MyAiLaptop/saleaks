SA LEAKS APP ICONS
==================

Place your SA Leaks logo/icon image here.

REQUIRED FILES:
- icon-192x192.png (192x192 pixels) - Main app icon
- icon-512x512.png (512x512 pixels) - Large app icon for splash screens

OPTIONAL FILES (for best quality on all devices):
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-384x384.png

FAVICON:
Also copy your icon to the public folder as:
- favicon.ico (for browser tab)

You can use a tool like https://realfavicongenerator.net to generate all sizes from one image.

RECOMMENDED:
- Use a square image with your SA Leaks logo
- PNG format with transparent background works best
- Make sure the icon is clear at small sizes (72x72)
